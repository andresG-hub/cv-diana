# CV - Diana Marcela Osorio Arias

Hoja de vida profesional premium optimizada para la vacante de **Analista de Costos e Inventarios**.

## Vista en vivo (GitHub Pages)

Visita: [https://andresG-hub.github.io/cv-diana/](https://andresG-hub.github.io/cv-diana/)

## Descargar PDF

1. Abre la [GitHub Page](https://andresG-hub.github.io/cv-diana/)
2. Haz clic en el boton "Descargar PDF" en la esquina inferior derecha
3. El PDF se generara automaticamente

## Agregar Foto

1. Coloca la imagen en la carpeta raiz del proyecto (ej: `foto-diana.jpg`)
2. En `index.html`, busca el bloque `<div class="photo-container">` y reemplaza el SVG por:

```html
<img src="foto-diana.jpg" style="width:100%; height:100%; object-fit:cover;" alt="Diana Marcela Osorio Arias">
```
